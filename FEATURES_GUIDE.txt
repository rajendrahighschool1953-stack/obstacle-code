PIN CONNECTIONS (Arduino Uno):
=================================================

Digital Pins:
- Pin 3: Servo (HC-SR04 tilt)
- Pin 4: Motor control (IN4)
- Pin 5: Motor control (IN3)
- Pin 6: Motor control (IN2)
- Pin 7: Motor control (IN1)
- Pin 8: HC-SR04 Trigger
- Pin 9: Motor PWM (ENA)
- Pin 10: Motor PWM (ENB)
- Pin 11: HC-SR04 Echo
- Pin 12: LED (obstacle indicator)
- Pin 13: Buzzer (audio alerts)

Analog Pins:
- Pin A5: Battery voltage (voltage divider)

=================================================
UPLOAD INSTRUCTIONS:
=================================================

1. Connect Arduino Uno via USB
2. Select Board: Tools > Board > Arduino Uno
3. Select Port: Tools > Port > COM[X]
4. Click Upload button
5. Open Serial Monitor (9600 baud) to see stats
6. Robot will beep on startup when ready!
